Este projeto é uma **aplicação demo** desenvolvida em **Java 21 + Spring Boot** para mostrar como integrar o Redis de forma simples em uma API REST.  

A aplicação expõe dois endpoints principais:  

### `POST /redis/salvar`  
Recebe `chave` e `valor` como parâmetros e os armazena no Redis.  

```http
POST /redis/salvar?chave=meuToken&valor=12345
→ Salvo com sucesso: meuToken -> 12345
```

### `GET /redis/buscar`

Recebe a chave e retorna o valor correspondente se existir.
```http
GET /redis/buscar?chave=meuToken
→ Valor: 12345
```
